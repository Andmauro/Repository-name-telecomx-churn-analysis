#  Telecom X - Análisis de Churn

Proyecto de análisis de datos realizado como parte del programa Oracle Next Education.

## Objetivo

Analizar los datos de clientes de Telecom X para identificar las causas de evasión de clientes (churn).

## Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

## Análisis realizados

- Extracción de datos desde API
- Limpieza de datos
- Análisis exploratorio
- Visualización de churn
- Análisis por tipo de contrato y cargos mensuales

## Principales insights

- Los clientes con contratos mensuales presentan mayor tasa de cancelación.
- Los clientes con cargos mensuales altos tienen mayor probabilidad de churn.
- Los contratos de largo plazo reducen la evasión.
